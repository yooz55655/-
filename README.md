# 校园网焚诀

## 介绍

在许多校园网络中，网络访问受到各种限制，如 DNS 劫持、流量限制、访问控制等。这个项目的目标是通过使用 OpenWRT 路由器，帮助用户绕过这些限制，提升校园网的使用体验。我们将介绍如何配置和优化 OpenWRT 路由器，使其适应不同的校园网络环境，并提供解决方案来应对常见的网络限制和局限性。

## 校园网认证破解解决方案 🚀

本项目旨在通过 **OpenWRT 路由器** 提供解决方案，帮助用户绕过不同类型的校园网认证方式，实现更自由的网络访问。校园网认证方式种类繁多，具体可以分为以下几种类型：

## 认证方式概述

根据认证的方式和技术实现，我们将解决方案分为三类：

1. **以校园网认证方式分类**：按照认证类型进行分类，例如 DNS 欺骗认证、强制登录门户、802.1X 认证等。每个认证类型有针对性的解决方案。
2. **以客户端认证还是网页认证分类**：根据认证方式的执行环境来分类。**客户端认证** 是通过设备（如路由器、终端）上的客户端程序来完成认证，**网页认证** 是通过浏览器完成的认证。
3. **以厂商和平台分类**：按照校园网认证使用的不同厂商平台进行分类，适配不同厂商的认证方式，如翼讯、锐捷、华为等。

每个分类下包含了相关的脚本、配置文件和教程，帮助用户快速解决特定认证问题。

## 解决方案分类

1. **[以校园网认证方式分类](solutions/authentication-methods/)**  
   根据认证的类型（如 DNS 欺骗、802.1X、强制登录门户）分类提供解决方案。

2. **[以客户端认证 vs 网页认证分类](solutions/client-vs-web/)**  
   根据认证的执行环境（客户端认证 vs 网页认证）进行分类。

3. **[以厂商和平台分类](solutions/vendor-platform/)**  
   针对不同厂商的校园网认证方式进行解决方案适配。

## 如何使用

- 请根据你的校园网认证方式选择相应的分类。
- 进入对应的文件夹，查看详细的解决方案、配置文件和安装说明。
- 如果你遇到任何问题，请提交 Issue 或加入讨论！
